# Font Assets

## Source Han Sans

The `SourceHanSans.ttc` file is the Source Han Sans CJK font family by Adobe.

**The font is NOT included in the repository** and must be downloaded:
- Open Settings > General in Overachiever
- Select "CJK (Source Han Sans)" as your font source
- Click the "Download Source Han Sans" button
- The app will automatically download and extract the font (~100 MB)
